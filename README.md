# Thai LLM Evaluation Kit v1.0

A lightweight benchmark and human evaluation framework for Thai-language Large Language Models (LLMs).

This project compares Thai responses generated by ChatGPT, Claude, and Gemini using a structured human annotation workflow inspired by public AI evaluation and responsible AI practices.

## Overview

As LLMs become increasingly capable of generating natural Thai text, evaluating response quality still requires human judgment. This kit provides a simple, reusable benchmark for comparing Thai-language responses across multiple models using consistent prompts, scoring criteria, and annotation guidelines.

## Models Compared

- ChatGPT
- Claude
- Gemini

## Objectives

- Compare Thai-language capabilities across leading LLMs.
- Build a reusable benchmark for Thai LLM evaluation.
- Demonstrate a structured human annotation workflow.
- Identify strengths and limitations of each model.
- Provide an open portfolio project for Thai NLP and AI evaluation.

## Files

### `prompts.csv`

A curated dataset of Thai evaluation prompts divided into three difficulty levels:

- Easy
- Medium
- Hard

Task categories include:

- Everyday conversation
- Translation
- Summarization
- Thai language and culture
- Reasoning
- Writing assistance

### `annotation_guideline.md`

Human evaluation guidelines designed specifically for Thai-language response assessment.

The rubric focuses on:

- Thai language naturalness
- Fluency
- Instruction following
- Helpfulness
- Safety
- Cultural appropriateness

### `evaluation_results.csv`

Evaluation results using the scoring rubric. Each record includes score fields for:

- Accuracy
- Instruction following
- Thai language quality
- Helpfulness
- Safety
- Overall score

## Evaluation Criteria

| Criterion | Description |
| --- | --- |
| Accuracy | Is the information factually correct and logically sound? |
| Instruction Following | Does the response satisfy the user's request and constraints? |
| Thai Language Quality | Is the Thai natural, fluent, grammatical, and appropriate for the context? |
| Helpfulness | Is the response useful, complete, clear, and actionable? |
| Safety | Does the response avoid harmful, inappropriate, or risky content? |

## Key Findings

The analysis focuses on qualitative differences across Thai-language tasks, including:

- Thai language naturalness
- Cultural appropriateness
- Translation quality
- Instruction-following consistency
- Overall response quality

### Objective Tasks

The three models generally perform similarly on tasks that require objective reasoning or structured output, such as:

- Translation
- Summarization
- Information extraction
- Classification
- Structured output
- Mathematical and logical reasoning
- Safety and responsible AI responses

Performance differences are usually minor for tasks with clear expected answers.

### Creative Language Tasks

Larger differences appear in tasks involving:

- Text generation
- Rewriting
- Recommendation and advice

These tasks require deeper understanding of Thai language nuance, emotional context, descriptive writing, and implicit meaning.

## Model Observations

### ChatGPT

Strengths:

- Balanced responses
- Reliable reasoning
- Consistent structure

Limitations:

- Sometimes less detailed than other models
- May sound less natural in Thai for some creative or nuanced tasks

### Gemini

Strengths:

- Strong understanding of Thai language nuance
- Rich and detailed explanations
- Strong reasoning ability

Limitations:

- Occasionally includes information that was not explicitly requested

### Claude

Strengths:

- Concise and clear responses
- Good understanding of natural Thai

Limitations:

- Sometimes less detailed
- May add unnecessary emojis in some responses

## Conclusion

All three models perform well on logical reasoning, factual accuracy, and structured-output tasks. The biggest differences appear in natural language generation, especially Thai fluency, context awareness, emotional expression, and cultural nuance.
